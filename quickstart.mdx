Follow these steps to set up your complete Agentic Team OS environment.

## Prerequisites

1. Install SST CLI globally:
```bash
npm install -g sst
```

2. Install Node.js 18+ and yarn

## Project Setup

1. Create your project directory:
```bash
mkdir my-agentic-project
cd my-agentic-project
```

2. Clone the template repositories:
```bash
# Clone Landing Page Template
git clone https://github.com/your-org/agentic-landing-page-template landing-page

# Clone Dashboard Template
git clone https://github.com/your-org/agentic-dashboard-template dashboard

# Clone API Template
git clone https://github.com/your-org/agentic-api-template api
```

## Environment Configuration

### Auth is Clerk
```bash
# Get these from Clerk Dashboard
CLERK_SECRET_KEY=
CLERK_PUBLISHABLE_KEY=
```

### API is Unkey
```bash
# Get these from Unkey Dashboard
UNKEY_API_KEY=
UNKEY_ROOT_KEY=
```

### Infrastructure is AWS
```bash
# Configure via AWS CLI or SST Console
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_REGION=us-east-1
```

### SST Cloudflare
```bash
# Get these from Cloudflare Dashboard
CLOUDFLARE_API_TOKEN=
CLOUDFLARE_ACCOUNT_ID=
```

### Payments is Stripe
```bash
# Get these from Stripe Dashboard
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET=
STRIPE_PRICE_ID=
```

## Template Configuration

### Landing Page (.env)
```bash
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/dashboard
NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/dashboard
```

### Dashboard (.env)
```bash
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
NEXT_PUBLIC_API_URL=http://localhost:3001
NEXT_PUBLIC_UNKEY_API_KEY=...
```

### API (.env)
```bash
PORT=3001
MONGODB_URI=mongodb://localhost:27017/agentic
REDIS_URL=redis://localhost:6379
UNKEY_API_KEY=...
OPENAI_API_KEY=...
STRIPE_SECRET_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...
```

## Starting Development

1. Start the API:
```bash
cd api
yarn install
sst dev
```

2. Start the Dashboard:
```bash
cd dashboard
yarn install
sst dev
```

3. Start the Landing Page:
```bash
cd landing-page
yarn install
sst dev
```

Your development environment will be available at:
- Landing Page: http://localhost:3000
- Dashboard: http://localhost:3002
- API: http://localhost:3001
