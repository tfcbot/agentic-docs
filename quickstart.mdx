# Quickstart Guide

Follow these steps to set up your complete Agentic Templates environment.

## Prerequisites

1. Install SST CLI globally:
```bash
npm install -g sst
```

2. Install Node.js 18+ and yarn

## Project Setup

1. Create your project directory:
```bash
mkdir my-agentic-project
cd my-agentic-project
```

2. Clone the template repositories:
```bash
# Clone Landing Page Template
git clone https://github.com/your-org/agentic-landing-page-template landing-page

# Clone Dashboard Template
git clone https://github.com/your-org/agentic-dashboard-template dashboard

# Clone API Template
git clone https://github.com/your-org/agentic-api-template api
```

## Template Configuration

### Landing Page (.env)
```bash
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/dashboard
NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/dashboard
```

### Dashboard (.env)
```bash
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...
NEXT_PUBLIC_API_URL=http://localhost:3001
NEXT_PUBLIC_UNKEY_API_KEY=...
```

### API (.env)
```bash
PORT=3001
MONGODB_URI=mongodb://localhost:27017/agentic
REDIS_URL=redis://localhost:6379
UNKEY_API_KEY=...
OPENAI_API_KEY=...
STRIPE_SECRET_KEY=sk_test_...
STRIPE_WEBHOOK_SECRET=whsec_...
```

## Starting the Development Environment

1. Start the API with SST:
```bash
cd api
yarn install
sst dev
```

2. Start the Dashboard with SST (in a new terminal):
```bash
cd dashboard
yarn install
sst dev
```

3. Start the Landing Page with SST (in a new terminal):
```bash
cd landing-page
yarn install
sst dev
```

SST will:
- Deploy infrastructure to your personal development stage
- Start local development servers
- Set up Live Lambda Development
- Create VPC tunnels as needed

Your development environment will be available at:
- Landing Page: http://localhost:3000
- Dashboard: http://localhost:3002
- API: http://localhost:3001
