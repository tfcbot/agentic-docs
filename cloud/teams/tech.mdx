The Tech Team is your AI-powered technical documentation and specification squad. They help you plan, document, and validate your internal tools and AI systems with detailed PRDs and technical specifications.

## What This Team Can Do

### Product Requirements (PRDs)
- Create comprehensive product requirement documents
- Define feature specifications
- Outline technical constraints
- Document user stories and flows
- Specify integration requirements

### Technical Specifications
- Develop detailed technical specifications
- Design system architectures
- Document API requirements
- Create data flow diagrams
- Specify security requirements

### AI System Design
- Plan AI integration strategies
- Document model requirements
- Specify data processing pipelines
- Define evaluation metrics
- Create testing frameworks

## How It Works

1. **Share Your Vision**: Describe your tool or system requirements
2. **Review Drafts**: Collaborate on specification documents
3. **Refine & Finalize**: Get production-ready documentation

## Example Use Cases

- Creating internal tool specifications
- Documenting AI system requirements
- Planning new feature implementations
- Designing system architectures
- Developing integration specifications
- Creating technical documentation
