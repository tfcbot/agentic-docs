# Adding New Agents

This guide walks you through the process of adding new AI agents to the Agentic Template system.

## Step 1: Define Agent Schema

Create a new schema file in the appropriate directory:

```typescript
// packages/core/src/agent-plane/your-agent/metadata/your-agent.schema.ts
export const YourAgentSchema = {
  type: 'object',
  properties: {
    input: {
      type: 'object',
      properties: {
        // Define input parameters
      }
    },
    output: {
      type: 'object',
      properties: {
        // Define expected output
      }
    }
  }
};
```

## Step 2: Create Agent Prompt

Define the agent's behavior and instructions:

```typescript
// packages/core/src/agent-plane/your-agent/metadata/your-agent.prompt.ts
export const YourAgentPrompt = `
# Role and Goal
You are [describe agent role]...

# Instructions
1. [Specific instruction]
2. [Another instruction]...
`;
```

## Step 3: Implement Use Cases

Create the main use cases for your agent:

```typescript
// packages/core/src/agent-plane/your-agent/usecases/create-strategy.usecase.ts
export class CreateStrategyUseCase {
  async execute(input: YourAgentInput): Promise<YourAgentOutput> {
    // Implementation
  }
}
```

## Step 4: Add Frontend Integration

1. Create agent service in the dashboard:
```typescript
// src/lib/agents/yourAgent.agent.ts
export class YourAgent {
  async createStrategy(input: YourAgentInput): Promise<YourAgentOutput> {
    // Implementation
  }
}
```

2. Add UI components as needed

## Step 5: Testing

1. Add unit tests for the agent
2. Add integration tests
3. Test the complete flow

## Best Practices

- Follow the existing architectural patterns
- Maintain consistent error handling
- Document all public interfaces
- Include comprehensive testing
- Keep the agent's responsibility focused and specific
